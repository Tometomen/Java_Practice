▶Static
class Student{
    int counter = 0;

    Student(){

        counter++;
    }
}

Student stu1 = new Student();
→オブジェクトを生成したとき
メモリの中にオブジェクトの領域が作られて
stu1の中にはそのオブジェクトのアドレス値が入る
stu1の中にcounterの領域が作られる
counterは初期値0
counter++なので入る値は1となる。


Student stu2 = new Student();
stu2もcounterの領域作られる。
counterは初期値0
counter++なので入る値は1となる。

何個インスタンスを生成しても
0が1にかわる処理が続くだけ

→1という値しか入らない

⇒インスタンス内の変数では無理！
⇒全インスタンスが共通して使える変数が必要。

▶Static変数,Staticメソッド

・全インスタンス変数が使えるメンバ変数やメソッドを定義するには【Static】を指定する。

・オブジェクトを生成していなくても利用できる。
⇒class名.変数[メソッド名]と記述

▶どんな時に使うのか？

①全部のインスタンス型共通して使う変数やメソッドを作りたい時に

②クラスの関連する便利なメソッドをまとめたいとき
